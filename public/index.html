🔥 Firebase Setup Guide ▼

Follow these steps to connect your app to Firebase (free) so your data syncs across all your devices.

1
Create a Firebase Project

Go to console.firebase.google.com → Click Add project → Enter a name (e.g. "my-expense-tracker") → Disable Google Analytics (optional) → Click Create project

2
Enable Email/Password Authentication

In Firebase Console → Authentication → Sign-in method → Enable Email/Password → Also enable Google if you want Google login → Click Save

3
Set Up Firestore Database

In Firebase Console → Firestore Database → Create database → Choose Production mode → Select a region (e.g. asia-south1 for India) → Click Enable

Then go to Rules tab and paste:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /users/{uid}/{document=**} { allow read, write: if request.auth.uid == uid; } } }
4
Enable Firebase Storage

In Firebase Console → Storage → Get started → Production mode → Click Done

Go to Rules and paste:

rules_version = '2'; service firebase.storage { match /b/{bucket}/o { match /receipts/{uid}/{allPaths=**} { allow read, write: if request.auth.uid == uid; } } }
5
Get Your App Config Keys

In Firebase Console → ⚙️ Project Settings → Your apps → Click Add app → Web ( icon) → Register app → Copy the firebaseConfig object values

apiKeyFIREBASE_API_KEY
authDomainFIREBASE_AUTH_DOMAIN
projectIdFIREBASE_PROJECT_ID
storageBucketFIREBASE_STORAGE_BUCKET
messagingSenderIdFIREBASE_MESSAGING_SENDER_ID
appIdFIREBASE_APP_ID
6
Deploy to Render (Free)

1. Push your code to GitHub (git init → git add . → git commit -m "init" → push to GitHub)

2. Go to render.com → New → Web Service → Connect your GitHub repo

3. It auto-detects render.yaml. Then in Environment Variables, add the 6 Firebase keys above

4. Click Deploy — your app goes live in ~2 minutes at yourapp.onrender.com

💡 Free tier: Render free plan spins down after 15 min inactivity. First load takes ~30 seconds. Upgrade to paid ($7/mo) for always-on.
7
Run Locally for Testing
cd "Tracker ERp" npm install node server.js # Opens at http://localhost:3000

Without Firebase env vars it runs in Demo Mode automatically.
